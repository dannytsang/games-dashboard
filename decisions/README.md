# Decisions

Architecture Decision Records (ADRs) for the games-dashboard project.

Use the format below when adding a new ADR. One ADR per substantive decision.

## ADR template

```markdown
# ADR-NNN: {Decision title}

- **Status:** Proposed | Accepted | Superseded by ADR-XXX
- **Date:** YYYY-MM-DD
- **Context:** What is the situation?
- **Decision:** What did we choose?
- **Consequences:** What becomes easier or harder?
- **Alternatives considered:** What else did we weigh?
```
